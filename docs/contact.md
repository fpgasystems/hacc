# Contact


If you encounter issues with accessing the cluster, have questions about the infrastructure or a feature request, feel free to reach out to us.

Write an email to `hacc@ethz.ch` and be sure to **use a fitting tag in the subject of your email** and always **mention your ETH shortname**. This allows us to help to you faster.

### Email Tags:
Write your emails with one of the following tags in your subject line, like this: `[Issue]: Can't reach hacc-build-01`.

- `[Issue]`: For issues regarding ETH Accounts, accessing the cluster or a specific server.
- `[Feature]`: For feature requests for the cluster (see note below).
- `[Info]`: For requesting info about the infrastucutre.
- `[Account]`: For requests regarding your ETH account, see [Account Renewal](./account-renewal.md).

> Feature requests should serve all users of the HACC. We therefore reject most requests for installing libraries/apps for someone's specific build process. We like to keep our OS clean and stable for all our users. So if you require a specific (version of a) library, be sure to use `venv` or `conda`, or build a development container using `docker`, or `apptainer`. Alternatively, you can always compile the library from scratch.
